# Agent Forge

AI-powered test generation agent that analyzes PR diffs and generates targeted test cases.

## Quick Start

```bash
pip install agent-forge
agent-forge run owner/repo --pr 42
```

## Development

```bash
git clone https://github.com/rkp4u/agent-forge.git
cd agent-forge
python3 -m venv .venv
source .venv/bin/activate
make dev
```
